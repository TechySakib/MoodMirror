# **App Name**: MoodMirror

## Core Features:

- Emotion Selection: Allow the user to select an emotion from a predefined grid of emotion options.
- Dynamic Verse Display: Display a relevant Qur'anic verse and its English translation based on the selected emotion. This feature also supports copying of the verses.
- Background transition: Use the selected emotion to change the app's background and theme, smoothly transitioning between calming colors and visuals.
- Generate Reflection Line: Use the selected emotion and selected verse, provide a single-line AI generated spiritual reflection appropriate to the verse that acts as a tool to focus on core theme.

## Style Guidelines:

- Primary color: Soft violet (#A099D9) to evoke a sense of calm and reflection.
- Background color: Light lavender (#E6E3F2), a desaturated version of the primary color for a subtle backdrop.
- Accent color: Pale rose (#D9A0A6), an analogous color providing gentle contrast and visual interest.
- Headline font: 'Poppins' (sans-serif) for emotion titles and prominent text; Body font: 'PT Sans' (sans-serif) for body text, like verse translations. The translation requires a highly legible and easy-to-read font, which PT Sans excels at.
- Employ emotion-specific animated emojis or icons to visually represent the user's selected feeling.
- Maintain perfect center alignment with a floating glass card as the main focus; ensure readability for both Arabic and English text.
- Utilize Framer Motion for gentle animations, like background color transitions and verse fade-ins, to create a spiritually calming experience.
